出现问题原因：JAI图像传输时丢包

解决方法：

使用 JAI Control Tool 采集图像 防止丢包

如何使用JAI Control Tool 采集图像

如何查看是否丢包？

其实观察刚开始采集图像时图片是否有撕裂的情况（有黑条条闪过）

如果丢包 设置相机据包为8000左右（我是8196）

如何设置相机数据包

设置相机数据包后用JAI Control Tool采图时黑屏，查看黑屏是是否有帧率

如何查看帧率

有帧率则查看调整网卡巨型帧

如何找到相机连接的网卡

网络和共享中心-以太网（就是相机那条网线）-属性-由勾选EBUS设置为不勾选EBUS 同时可以看到连接时使用的网卡

找到网卡右击-属性-高级-巨型帧（由关闭设为9KB MTU）接收缓冲区最好设为2048（我的最大只有512，只能这样了）

提醒：文档中指出接收缓冲区最好为2048，而我的网卡最大才512，图中说设为最大值不太明确，请结合自己的实际情况

是不是我的网卡太菜了，众筹给我买个好的可以吗？？？

如果仍丢包则根据文档设置延时直至成功

如何设置延时？

参照文档设置延时（主要是帧率）我的Max Fps是2.00 ，Band Width是90%

接下来查看是否丢包 如有丢包可以再次调整数据包大小和延迟
